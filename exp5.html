Study of nested for-loop

Nested For-loop

In many cases we may use loop statement inside another looping statement. This type of looping is called nested loop. In nested loop the inner loop is executed first and then outer. The nested loop must be used to input or output multi-dimensional array elements.

Algorithm:

Step 1: Read number of rows as input (i.e. n=4)
Step 2: for(i=1;i<=n;i++) Step 3: i defines no.of rows Step 4: for(j=1;j<=i-1;j++)
Step 3: j defines no. of spaces for a line
Step 4: for(k=65;k<=65+n-i;k++)
Step 5: here k is the character to print the character for above condition.
Step 6: for(k=65+n-i-1;k>=65;k--)
Step 7: here again character is printed
Step 8: above process continues till loop terminate.
Step 9: new line.
