Experiments

Here are some experiments to help you clarify your concepts of C in a visual manner.

Write a program to accept the temperature in Celsius and to convert and display it in Fahrenheit

Write a program to accept three numbers and display largest of three using a nested if else statement

Write a program to implement an arithmetic calculator for addition,subtraction, multiplication, division and modulo operation using switch case

Write a program to check whether an entered number is prime number or not using for-loop

Write a program to generate the following pattern using nested for-loop

Write a program to check whether the given number is armstrong number or not using while loop

Write a program to find binary equivalent of a given decimal number using do-while loop

Write a program to find value of BIO using function, where BIO is defined as BIO= n!/(r!(n-r)!), n and r are natural numbers

Write a program to check whether entered string is palindrome or not

Write a program to swap two numbers using call by address

How do I benefit from these experiments?

Simulation.

Each one of these experiments are equipped with a simulation that imitates how a code looks like when you are in real time dealing with it and gives you a general
idea of how programming looks like.These experiments start from beginner level and get progressively tougher as per the user pace.

Topic Oriented Theory.

These experiments will each be provided with the required theory essential to the given problem statement. This provides a diversified way of looking at problems one topic at
a time.

Posttest.

All the experiments have a common trope where each
experiment is followed by a quick and hands on
quiz that has questions related to the experiment,
further making you more familiar to the topic and
make you the champ at C Programming!
